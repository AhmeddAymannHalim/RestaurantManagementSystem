# Restaurant Management System - Frontend

## 🚀 Quick Start

1. Open `index.html` in your browser
2. Click "Launch Dashboard" to access the management system
3. Default credentials:
   - Username: `admin`
   - Password: `Admin@123`

## 🌐 Language Support

- English (EN)
- Arabic (ع)

Toggle language using buttons in top-right corner.

## 📚 API Documentation

Access Swagger API docs at: https://localhost:7054/swagger

## 🎯 Features

- Multi-language support (EN/AR)
- Real-time order tracking
- Table management
- Menu management
- User authentication
- Role-based access control

## 🔧 Backend Setup

The backend API must be running at `https://localhost:7054`

## 📦 Deployment

This is a static frontend that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

**Note:** Update `API_BASE_URL` in `app.js` to your production API URL before deployment.
